    private getMelodiesCore(chordUnit: ChordUnit, melodyList: string[], options: Object): string[] source

    Recursive function that adds new notes to the previous notes passed into it. On the first call of this function, melodyList should be null.

    Params:

    NameTypeAttributeDescription
    chordUnit ChordUnit

    the ChordUnit for the next chord
    melodyList string[]
    • nullable: true

    list of existing melodies
    options Object

    if true, generated melodies will be filtered
    options.filtered boolean

    if true, generated melodies will be filtered
    options.rawOutput boolean

    if true, generated melodies will be given as strings

    Return:

    string[]

    a list of melodies. Each element is a string represeting a melody. Each melody string is written as a series of pitches delimited by a space.
